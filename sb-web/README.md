# Sailboat Web

Web UI for the Sailboat application.

## Technologies

1. React + Typescript
2. Chakra UI
3. React Select
4. React Toastify
5. Axios
6. React Icons

## How to run

- Install Node 16.2.0

### Development 

```
$ git clone https://github.com/hrishikeshpaul/sailboat.git
$ cd sb-web
$ yarn install
$ yarn start
```

### Linting

```
$ yarn lint
```

### Build for production

```
$ yarn build
```
